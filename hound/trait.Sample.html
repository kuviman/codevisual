A type that can be used to represent audio samples.

Via this trait, decoding can be generic over i8, i16 and i32. All bit depths up to 32 bits per sample can be decoded into i32, but it takes up more memory. If you know beforehand that you will be reading a file with 16 bits per sample, then decoding into an i16 will be sufficient.
